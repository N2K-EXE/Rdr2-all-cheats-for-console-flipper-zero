# RDR2 Cheats BadKB

Unlocks all Red Dead Redemption 2 cheat codes automatically with Flipper Zero BadKB. No manual typing needed.

## Important

After each cheat, reopen **Enter Cheat Code** and return to the input screen.
Press the Flipper button only when the input box is open.

## Download

[Download Script](./rdr2-all-cheats.txt)

## Usage

1. Open the cheat menu in RDR2
2. Run the script on Flipper Zero
3. After each code, reopen cheat input
4. When the input box is ready, press the Flipper button

## Requirements

* Flipper Zero
* USB-A TO USB-C cable
* Red Dead Redemption 2
